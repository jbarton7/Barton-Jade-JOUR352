2.  Describe the purpose of a DIV tag.

A DIV tag divides content into blocks, grouping sections of HTML together. DIVS are useful because they allow you to style all of the elements within the DIV element specifically.

4.  There are two main "areas" or "sections" in the basic structure of an HTML page. In which area would you put a link to a stylesheet?

5.  In which main "area" or "section" of an HTML page would you put text, images or DIV tags?

10.  Why do we use media queries?

We use media queries in CSS in order to change the styles of elements when displayed on different screen sizes. Media queries are essential for responsive design so elements can be styled on a variety of screen sizes.


11. What is the main difference between a serif and a sans-serif font?

The main difference between serif and sans-serif is that with serif font there are additional lines that extend off of the ends of the letter.
